El Capitan
==========

El Capitan is the CORAL2 flagship supercomputer at Lawrence Livermore National Laboratory (LLNL), commissioned in 2024. It is an HPE Cray system designed for large-scale simulation and modeling, featuring AMD EPYC CPUs and AMD MI300A APUs (GPU+CPU). El Capitan is deployed in the Secure Computing Facility (SCF) and is a primary platform for the Advanced Simulation and Computing (ASC) program.

CTS-2
=====
